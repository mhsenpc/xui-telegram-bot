# xui-telegram-bot

# معرفی اجمالی
این نرم افزار یک ربات تلگرام است که به شما امکان استفاده از پنل XUI می دهد

شما با استفاده از این ربات تلگرامی می توانید اقدام به فروش اکانت های VLESS, VMESS کنید
